# Advanced Student Management System
## Description 
Aconsole-based Student Management System developed in C++ to manage student records efficiently.
## Features
- Add New Student
- View All Students
- Search Student by Roll Number
- Update Student Details
- Delete Student Record
- Display Class Topper
- Count Total Students
- Calculate Statistics (Average, Highest, Lowest Marks)
- Automatic Grade Assignment
## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- STL Vector
- File Handling
## Learning Outcomes
- OOP Concepts
- Data Management
- File Handling
- CRUD Operations
- STL Usage
